# Project Setup Guide

This project is configured for both JavaScript and PHP development.

## Directory Structure

- **`vue/`**: This directory is dedicated to creating and managing Vue projects related to **Vue+**.
- **Root**: Contains configuration files and common utilities.

## How and what is vue folder
Created and used to store all vue+ project.
```bash
npm create vue@latest
```
- Components preinstalled
   - pinia
   - vue-route
   - linter
- Setup instruction for *vue* folder
   ```bash
   npm install
   npm run dev
   ``` 



## Setup Instructions

1. **Prerequisites**:
   - Node.js (latest LTS)
   - PHP 8.x
   - Composer

2. **Installation**:
   - For JavaScript dependencies:
     ```bash
     npm install
     ```
   - For PHP dependencies:
     ```bash
     composer install
     ```

3. **Vue+ Projects**:
   - Navigate to the `vue` folder to start new Vue+ related projects.
